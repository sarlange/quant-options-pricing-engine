# quant-options-pricing-engine
A Python quantitative finance toolkit for pricing options, calculating Greeks, estimating implied volatility, and running Monte Carlo simulations.

# Quant Options Pricing Engine

A quantitative finance project that implements:

- Black-Scholes Option Pricing
- Option Greeks
- Monte Carlo Simulation
- Interactive Streamlit Dashboard

## Features

### Black-Scholes Model
Calculates theoretical European option prices.

### Greeks
Measures option sensitivity:
- Delta
- Gamma
- Vega
- Theta
- Rho

### Monte Carlo Simulation
Simulates thousands of future stock price paths to estimate option value.

## Technologies

- Python
- NumPy
- SciPy
- Streamlit
